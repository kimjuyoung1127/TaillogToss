# settings/ — 설정 페이지 전용 컴포넌트

`/settings` 라우트에서만 사용하는 UI 조각을 보관한다.

## 파일

| 파일 | 용도 |
|---|---|
| `SettingsSectionCard.tsx` | 섹션 제목 + 카드 컨테이너 |
| `SettingsToggleRow.tsx` | 스위치 토글 행 |
| `SettingsNavRow.tsx` | 네비게이션/액션 행 |
| `SettingsStatusRow.tsx` | 저장 상태 배지 행 |
| `SettingsPermissionBanner.tsx` | 권한 안내 배너 |
| `SettingsStepperRow.tsx` | 시간 증감 행 |
| `SettingsDivider.tsx` | 설정 카드 내부 구분선 |
| `SettingsScreenSkeleton.tsx` | 설정 로딩 스켈레톤 |
| `SettingsScreenError.tsx` | 설정 에러 상태 |
| `NotificationSettingsSection.tsx` | 알림 채널/타입/방해 금지 시간 섹션 |
| `AiCoachingSettingsSection.tsx` | AI 코칭 톤/관점 섹션 |
| `AccountSettingsSection.tsx` | 프로필/약관/개인정보 진입 섹션 |
| `ServiceSettingsSection.tsx` | 구독/반려견 전환 진입 섹션 |
